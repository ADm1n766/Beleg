first line - jumping / falling animation (bird is flapping his wings)
second line - walking animation (feet are moving)
third line - eating or ducking animation

Image is in SVG vector format so you can scale character to any bitmap size you need. Created in Inkscape, which can be used to convert image to bitmap.
